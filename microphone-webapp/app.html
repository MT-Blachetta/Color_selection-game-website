Willkommen bei unserem psychologischem Test

Es werden nacheinander Wörter von Farben gezeigt und
die Aufgabe ist so schnell wie möglich den Namen ins Mikrophon zu sprechen
grün, rot, grau, violett
Sobald die Antwort erkannt wurde erscheint das nächste Wort
Benennen sie alle korrekt wobei die Zeit gestoppt wird -
Am Ende wird gezeigt wie schnell sie waren,
klicken sie auf "Ausprobieren" für ein Beispiel

Bevor sie fortfahren stellen sie fest ob ihr Mikrophon an ist und vom Browser aktiviert werden kann,
geben sie die notwendigen Berechtigungen damit die Seite das Mikrophon verwenden kann.
